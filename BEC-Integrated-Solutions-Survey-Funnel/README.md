# BEC Integrated Solutions Survey Funnel

A survey-style Click Funnel concept for BEC Integrated Solutions focused on wireless nurse call systems.

## Pages

- `index.html` - Survey funnel and comparison sections
- `recommendations.html` - Personalized recommendation summary
- `checkout.html` - Quote/checkout redirect page

## Theme

Built around a BEC-style healthcare/industrial palette using navy, blue, red, gold, and clean slate neutrals.

## Content Sources

Uses the BEC wireless nurse call kits product page content and feature highlights provided by the user.
